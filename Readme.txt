Приложение Quiz реализовано на Python v3.8.8. и фремворке Django v3.2.1.

Приложение позволяет пользователю пройти тестирование.

Тест имеет 5 вопросов.
В приложении есть следующие методы:
	main - отвечает за перенаправление на главную страницу и обнуление старых ответов пользователя в базе данных
	save_user_answers - отвечает за сохранение выбранных пользователем вариантов ответа в базу данных, а так же изменение их значения(при возврате на предыдущий вопрос)
	questions - отвечает за перенаправление на страницу с вопросами, а так же генерацию списка вопросов. Так же он вызывает метод get_result и перенаправляет на страницу, на которую выводится результат тестирования
	get_result класса QuizResultService - вычисляет результат тестирования
Приложение автоматически генерирует в шаблонах html либо а)форму с type=checkbox либо б)форму с type=radio в зависимости от количества правильных вариантов ответа
Метод get_result при наличии нескольких правильных вариантов ответа считает ответ правильным только в том случае, если все ответы пользователя на этот вопрос совпали со списком правильных ответов
В приложении реализован функционал, позволяющий возвращаться на предыдущий вопрос и менять свой ответ, уже сохраненный в базе данных.
При реализации фронтенда был использован Bootstrap
